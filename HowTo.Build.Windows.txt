
Host To Build on MS Windows
---------------------------

1. Requirements:
1.1. MS Visual Studio Express (or better) 10:
     Free install: http://www.microsoft.com/visualstudio/en-us/products/2010-editions/visual-cpp-express
1.2. Boost C++ libraries 1.51 32-bit.
     Variants:
         1) Multithreaded
         2) Multithreaded debug
     Libraries:
         1) Chrono
         2) DateTime
         3) Exceptions
         4) Filesystem
         5) Python
         6) Signals
         7) System
         8) Thread
     Official site: http://www.boost.org/
     Free install (ready builds): http://www.boostpro.com/download/
1.3. Python 2.7 32-bit.
     http://www.python.org/download/
1.4. Protocol Buffers 2.4.1.
     Free compiler for Windows: http://code.google.com/p/protobuf/downloads/list/
1.5. Qt
     Free sources and binaries: http://qt-project.org/downloads
1.6  VMime 0.9.1
     Free sources: http://www.vmime.org/downloads.html
     Compilded by Visual Studio 2010 binaries: http://download.robotdk.com/externals/vmime_vs2010_0.9.1.bz2

2. Setup:
2.1. Create or change envelopment variable BOOST_DIR_X86 - set path to
     installed Boost. Path should point to root install directory, without
     version. Ex.: if Boost installed into
     "c:\Program Files (x86)\Boost\boost_1_47\", BOOST_DIR_X86 should has
     value "c:\Program Files (x86)\Boost\".
2.2. Unpack Protocol Buffers compiler. Add path to protoc.exe to the PATH
     envelopment variable.
2.3. Install Qt.
2.4. Unpack VMime archive into "externals\VMime\" as it distributed in
     archive (path to sources will looks like
     "externals\VMime\libvmime-x.x.x\"). Build library if you use archive
     with sources.

3. Build:
3.1. Open "sources\RobotDK.sln" with Visual Studio and build it for
     required configurations.
3.2. Result will be in "output\x86\bin\".
